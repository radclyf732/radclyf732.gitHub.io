故事开端

On this page
  • 参考

2017年10月31日,迅雷旗下全新一代共享经济智能硬件——玩客云战略发布会在北京举办，当天迅雷CEO、网心科技CEO陈磊宣布，迅雷将自玩客云开始，全面迈入“共享计算”全新战略布局时代。陈磊称，未来迅雷将持续加大对共享计算的探索，并将在为B端企业客户提供服务的同时，向所有普通个人用户开放共享计算服务1。

如果当时的你买了一台玩客云将它连接硬盘加入链克计划，那么你的设备加硬盘就相当于一个小型的CDN服务器，它会接受相应的下载请求或网络缓存请求，这些缓存将为你周围的上网用户提供网络加速。

当然这些下载或缓存任务是由迅雷网心提供的，网心科技将用户提供的CDN资源进行整合打包出售给企业用户（XX直播，XXTV），而作为提供CDN的用户可以得到相应的代币（链克）作为报酬，而这些代币就是基于区块链技术（不可增发，无法伪造）生成的，用户可以选择将其进行出售或交换。

这可以说是一个互利的模式：用户提供限制的网络带宽获取代币，网心通过将用户提供的CDN资源整合进行出售2。

这台迅雷发布的矿机，可以作为个人存储使用。定价599，还一度供不应求，最疯狂的时候黄牛二手炒到2000多。

后期就是逐渐凉凉，先是跟“子公司”迅雷金融（已改名）撕了一架，导致币价下挫，迅雷网心被互金协会点名指出其实在变相进行ICO，网心科技随即对链克进行“自宫”，关闭了链克在国内的提币功能2。

时间慢慢过去，大批玩家退出，大量二手设备流入市场。终于，在2024年1月2日时玩客云官网也发布了公告，宣布于同年2月29日停止APP支持3。宣告其作为“个人存储”也凉到透底。

大批低价的设备到越来越多人手里，依靠社区和开源的力量，这款机器被开发了许多用途：轻NAS、电视盒子、旁路由、打印服务器等等。虽然性能有限，

但是它的价格只需50多！啥都能玩！#
