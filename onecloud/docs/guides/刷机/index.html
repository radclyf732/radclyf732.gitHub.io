刷机

On this page
  • 电脑端准备
  • 连接

虽然玩客云服务早已凉凉，但玩客云的原版系统还留有一些挖矿留下的后门。我之前使用过原版系统一段时间，发现它会时不时访问硬盘目录，过一会硬盘灯就会狂闪，使得硬盘不能很好休眠，同时在目录里面留下一些隐藏的垃圾文件夹。要获得良好的使用体验，必定给他全部刷掉。

在网上有大佬给出了许多教程，这次打算刷个Armbian，部署CasaOS，在docker里面挂个Alist，再挂个小雅，作为局域网的娱乐中心使用。因为刷机是Linux+docker环境，可以实现的软件功能与大几千的Nas齐平。

电脑端准备#

1.首先下载armbian固件，在Github下载老的6.1.9固件。

2.下载安装烧录工具Amlogic_USB_Burning_Tool

连接#

1.用双Type-A头数据线连接电脑。注意要插玩客云接近HDMI口的USB口才能识别

也可以用手机充电线插笔记本Type-C，另一头插玩客云

2.找到板子背面的两个触点，短接后上电

3.若无意外，上电后你将会听到清脆的

“噔，噔噔噔”象征着USB接入Windows的提示音

同时刷机软件的进度条开始roll

故障排除(๑˃̵ᴗ˂̵)

如果反反复复都提示控制命令出错、读取DDR卡着不动、停止报错：可以换一个USB口、换根线、最后可以果断尝试台式电脑的后置USB。我就是笔记本死活刷不上，换台式机一下就好了。

4.等待刷完看到这样就是成功了，可以把线拔下来，重新给玩客云上电开机，插上网线准备下一步安装系统操作。
